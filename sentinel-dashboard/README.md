# Sentinel Dashboard

## Command

```shell
## start sentinel dashboard
$ docker compose -f docker-compose.yml -p sentinel_dashboard up -d

## stop sentinel dashboard
$ docker compose -p sentinel_dashboard down --rmi local -v
```
